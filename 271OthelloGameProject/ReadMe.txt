1. locate the path \271OthelloGameProject\x64\Release\271OthelloGameProject.exe to run the program.
2. the first step is to choose what color of player you want to be.
3. each turn the program will ask the user to input a pair of number to locate the cell that the player wish to place the tile.
4. input format: # \space\ #£¬with the first number is the column number, the second is the row number
